MAIL -> boolean delbcc ( [ string address ] )


Return TRUE if has been successfully deleted, FALSE if not.

string address format is "localpart@domainpart.tld".

Example:






$m = new MAIL;

// add Bcc mail address
$m->AddBcc('localpart@domainpart.tld');

// delete Bcc mail address
$m->DelBcc('localpart@domainpart.tld');

// delete all Bcc mail addresses
$m->DelBcc();










See Also: MAIL->AddBcc()
