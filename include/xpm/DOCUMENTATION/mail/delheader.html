MAIL -> boolean delheader ( [ string name ] )


Return TRUE if has been successfully deleted, FALSE if not.

Example:






$m = new MAIL;

// add headers
$m->AddHeader('X-Whatever', 'first value');
$m->AddHeader('X-Whatever', 'second value');

// delete only "X-Whatever" header name entries
$m->DelHeader('X-Whatever');

// delete all headers
$m->DelHeader();










See Also: MAIL->AddHeader()
