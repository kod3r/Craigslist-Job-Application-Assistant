MAIL -> boolean delcc ( [ string address ] )


Return TRUE if has been successfully deleted, FALSE if not.

string address format is "localpart@domainpart.tld".

Example:






$m = new MAIL;

// add Cc mail address
$m->AddCc('localpart@domainpart.tld');

// delete Cc mail address
$m->DelCc('localpart@domainpart.tld');

// delete all Cc mail addresses
$m->DelCc();










See Also: MAIL->AddCc()
