MAIL -> boolean path ( string address )


Return TRUE if has been successfully set, FALSE if not.

Set "Return-Path" mail header value for receiver.

string address format is "localpart@domainpart.tld".

Example (verification):






$m = new MAIL;

// set "Return-Path"
$m->Path('localpart@domainpart.tld');







Example (deprecated):






$m = new MAIL;

$m->Path = 'localpart@domainpart.tld';










See Also: MAIL->From(), MAIL->AddHeader()
