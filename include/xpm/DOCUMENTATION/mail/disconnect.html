MAIL -> boolean disconnect ( [ resource connection ] )


Return TRUE if has been successfully disconnected, FALSE if not.

Send "QUIT" SMTP command and close resource connection(s).

If resource connection is not provided will disconnect all open connections.

Example:






$m = new MAIL;

// first connection
$c1 = $m->Connect('smtp.hostname1.tld');
// second connection
$c2 = $m->Connect('smtp.hostname2.tld');

// disconnect first connection
$m->Disconnect($c1);

// or, disconnect all open connections
$m->Disconnect();










See Also: MAIL->Connect(), SMTP::Disconnect()
