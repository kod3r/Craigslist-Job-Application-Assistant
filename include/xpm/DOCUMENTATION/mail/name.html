MAIL -> boolean name ( string host )


Return TRUE if has been successfully set, FALSE if not.

Set hostname (can be IPv4 address) used in EHLO/HELO SMTP conversation.

Example (verification):






$m = new MAIL;

// set hostname
$m->Name('localhost');

// send mail to MX zone
$m->Send('client');







Example (deprecated):






$m = new MAIL;

$m->Name = 'localhost';










See Also: MAIL->Connect()
