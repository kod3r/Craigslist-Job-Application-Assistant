MAIL -> boolean addbcc ( string address )


Return TRUE if has been successfully added, FALSE if not.

string address format is "localpart@domainpart.tld".

Example (verification):






$m = new MAIL;

// add "Bcc" mail address
$b = $m->AddBcc('localpart@domainpart.tld');

echo $b ? 'added' : 'error';







Example (deprecated):






$m = new MAIL;

// add "Bcc" mail address
$m->Bcc[] = 'localpart@domainpart1.tld';
// add another "Bcc" mail address
$m->Bcc[] = 'localpart@domainpart2.tld';










See Also: MAIL->DelBcc(), FUNC::is_mail()
