MAIL -> boolean subject ( string content [, string charset [, string encoding ]] )


Return TRUE if has been successfully set, FALSE if not.

Default values are:
string charset: utf-8
string encoding: quoted-printable

string encoding possible values are: quoted-printable or base64

Example (verification):






$m = new MAIL;

// set "Subject" mail header value
$s = $m->Subject('Hello World!');

echo $s ? 'success' : 'error';







Example (deprecated):






$m = new MAIL;

// set "Subject" mail header value, charset and encoding
$m->Subject = array(
'content'  => 'Hello World!', // required
'charset'  => 'utf-8', // optional
'encoding' => 'base64' // optional
);










See Also: MAIL->AddHeader()
