MAIL -> boolean priority ( [ mixed level ] )


Return TRUE if has been successfully set/unset, FALSE if not.

mixed level possible values are: integer 1, 3, 5; string High, Normal, Low

mixed level equivalent values are:
 integer 1 = string High
 integer 3 = string Normal
 integer 5 = string Low

Example (verification):






$m = new MAIL;

// set priority to "Normal" or "3"
$m->Priority(3);

// set priority to "High" or "1"
$m->Priority('High');

// unset priority
$m->Priority();







Example (deprecated):






$m = new MAIL;

// set priority to "High" or "1"
$m->Priority = 1;

// unset priority
$m->Priority = null;










See Also: MAIL->AddHeader()
