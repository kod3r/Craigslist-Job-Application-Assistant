POP3 :: boolean pdele ( resource connection, integer message )


Return TRUE if has been successfully set, FALSE if not.

Send "DELE" POP3 command and mark the integer message with "delete" flag.

Notice: afther, you must send "QUIT" POP3 command to delete the integer message from server.




See Also: POP3::pQuit(), POP3::Disconnect(), POP3::pRset()
