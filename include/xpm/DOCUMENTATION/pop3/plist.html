POP3 :: mixed plist ( resource connection [, integer message ] )


Return ARRAY if has been successfully sent, FALSE if not.

Send "LIST" POP3 command. Get a scan listing of one or all messages.

Notice: the ARRAY result contain only integer values.
integer message => integer bytes.

Example:






// connect to 'pop.hostname.tld'
$c = POP3::Connect('pop.hostname.tld', 'username', 'password') or die(print_r($_RESULT));

// get the list of messages
$l = POP3::pList($c) or die(print_r($_RESULT));
// or, specify only the old message
// $l = POP3::pList($c, 1) or die(print_r($_RESULT));

// disconnect from server
POP3::Disconnect($c);

// print result
print_r($l);










See Also: POP3::pStat(), POP3::pUidl()
