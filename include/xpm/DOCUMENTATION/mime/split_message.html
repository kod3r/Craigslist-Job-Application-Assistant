MIME :: array split_message ( string message )


Return multidimensional ARRAY ('header' => string headers, 'content' => string body).

Example:






// standard mail message RFC2822
$m = 'From: me@addr.net'."\r\n".
     'To: my@addr.com'."\r\n".
     'Subject: test'."\r\n".
     'Content-Transfer-Encoding: base64'."\r\n".
     'Content-Type: text/plain; charset="ISO-8859-1"'."\r\n\r\n".
     base64_encode('Text message.');

$r = MIME::split_message($m);

// print result
print_r($r);










See Also: MIME::split_mail()
