MIME :: string encode_header ( string content [, string charset [, string encoding [, integer line_length [, string line_end ]]]] )


Return STRING encoded.

Default values are:
string charset: utf-8
string encoding: quoted-printable
integer line_length: 52
string line_end: CRLF [\r\n]

string encoding possible values are: quoted-printable or base64

Example:






echo MIME::encode_header('hello world', 'iso-8859-1', 'base64');










See Also: MIME::decode_header()
