FUNC :: boolean is_hostname ( string hostname [, boolean have_address ] )


Return TRUE if is a public hostname format, FALSE if not.

boolean have_address default value is: false

Example:






// test if "domain.net" is hostname format and have IP address
echo FUNC::is_hostname('domain.net', true) ? 'true' : 'false';










See Also: FUNC::is_ipv4()
